PyBioMed has been successfully tested on Linux and Windows systems. After installing RDKit and pybel successfully, The author could download the
PyBioMed package via: https://github.com/gadsbyfly/PyBioMed/blob/master/doc/download/PyBioMed-1.0.zip. The install process of PyBioMed is very easy:

Pybel can be downloaded via: https://openbabel.org/docs/dev/UseTheLibrary/Python_Pybel.html

RDkit can be downloaded via: http://www.rdkit.org/

On Windows:

(1): download the PyBioMed-1.0.zip

(2): extract or uncompress the PyBioMed-1.0.zip file

(3): cd PyBioMed-1.0

(4): python setup.py install

On Linux:

(1): download the PyBioMed package (.zip)

(2): unzip PyBioMed-1.0.zip

(3): cd PyBioMed-1.0

(4): python setup.py install or sudo python setup.py install

The online version of the documentation is available here:
http://pybiomed.readthedocs.io/en/latest/

Some information about using the software from Python is in the
userguide.rst in Docs
